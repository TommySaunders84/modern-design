Design System Documentation

Welcome to our comprehensive design system documentation. This guide will help you understand and implement our design principles, components, and best practices.

Typography System

Our typography system is built around system fonts, chosen for their excellent readability and modern aesthetic across all platforms.

Font Stack

font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'Roboto', 'Oxygen', 'Ubuntu', 'Cantarell', sans-serif;

The primary typeface uses system fonts with careful fallbacks ensuring consistent rendering across all platforms.

Font Weights

  • Light (300) - Used for large headings and decorative text
  • Regular (400) - Default body text weight
  • Medium (500) - Navigation links and secondary headings
  • Semibold (600) - Primary headings and emphasis
  • Bold (700) - Major headings and high emphasis text

Color System

Our color palette is built around a semantic approach, using CSS custom properties for easy theming and consistency.

Primary Colors

  • Background: #ffffff - Pure white for clean, minimal aesthetic
  • Foreground: #09090b - Near-black for high contrast text
  • Muted: #f4f4f5 - Light gray for subtle backgrounds
  • Border: #e4e4e7 - Medium gray for borders and dividers

Component Architecture

Our components follow modern principles with utility-first CSS and consistent naming conventions. Each component is designed to be composable, accessible, and easy to customize.

Design Principles

  • Consistency - Uniform spacing, colors, and typography across all components
  • Accessibility - WCAG 2.1 AA compliance with proper focus states and ARIA attributes
  • Flexibility - Components accept custom classes and can be easily themed
  • Performance - Minimal CSS with efficient selectors and no unused styles

Mobile Optimization

The design system includes comprehensive mobile optimization with responsive navigation, touch-friendly interactions, and mobile-first design patterns.

Mobile Features

  • Responsive Navigation - Collapsible hamburger menu on mobile devices
  • Touch Interactions - Larger touch targets and gesture-friendly components
  • Flexible Typography - Fluid font sizes using clamp() for optimal readability
  • Adaptive Layouts - Grid systems that stack vertically on smaller screens

Design Tokens

Design tokens are the visual design atoms of the design system. They are named entities that store visual design attributes.

Color Tokens

All color values are stored as CSS custom properties for easy theming and consistency.

:root { --background: #ffffff; --foreground: #09090b; --muted: #f4f4f5; --muted-foreground: #71717a; --border: #e4e4e7; --primary: #18181b; --primary-foreground: #fafafa; --secondary: #f4f4f5; --secondary-foreground: #18181b; --accent: #f4f4f5; --accent-foreground: #18181b; --destructive: #ef4444; --destructive-foreground: #fafafa; --success: #22c55e; --warning: #f59e0b; --ring: #18181b; }

Spacing Tokens

Consistent spacing scale based on a 4px base unit for harmonious layouts.

:root { --space-1: 0.25rem; /* 4px */ --space-2: 0.5rem; /* 8px */ --space-3: 0.75rem; /* 12px */ --space-4: 1rem; /* 16px */ --space-5: 1.25rem; /* 20px */ --space-6: 1.5rem; /* 24px */ --space-8: 2rem; /* 32px */ --space-10: 2.5rem; /* 40px */ --space-12: 3rem; /* 48px */ --space-16: 4rem; /* 64px */ --space-20: 5rem; /* 80px */ --space-24: 6rem; /* 96px */ }

Typography Tokens

Fluid typography scale that adapts to screen sizes for optimal readability.

:root { --font-size-xs: 0.75rem; /* 12px */ --font-size-sm: 0.875rem; /* 14px */ --font-size-base: 1rem; /* 16px */ --font-size-lg: 1.125rem; /* 18px */ --font-size-xl: 1.25rem; /* 20px */ --font-size-2xl: 1.5rem; /* 24px */ --font-size-3xl: 1.875rem; /* 30px */ --font-size-4xl: 2.25rem; /* 36px */ --font-size-5xl: 3rem; /* 48px */ --font-size-6xl: 3.75rem; /* 60px */ }

Shadow Tokens

Elevation system using layered shadows for depth and hierarchy.

:root { --shadow-sm: 0 1px 2px 0 rgba(0, 0, 0, 0.05); --shadow: 0 1px 3px 0 rgba(0, 0, 0, 0.1); --shadow-md: 0 4px 6px -1px rgba(0, 0, 0, 0.1); --shadow-lg: 0 10px 15px -3px rgba(0, 0, 0, 0.1); --shadow-xl: 0 20px 25px -5px rgba(0, 0, 0, 0.1); }

Border Radius Tokens

Consistent border radius values for cohesive component styling.

:root { --radius: 0.5rem; /* 8px */ }
